# Telegram-бот «Фонетист»

## Описание
Это бот, который отвечает на вопросы по курсу «Сегментные единицы» на основе базы знаний, сформированной из экзаменационных билетов.

## Функции
- прием текстовых вопросов в свободной форме;
- анализ запроса пользователя;
- поиск релевантной информации в базе знаний;
- генерация развернутого теоретического ответа с помощью LLM.

## Стек технологий
- Python
- Telegram Bot API
- aiogram
- LLaMA / аналогичная LLM
- Markdown (.md)

## Структура проекта
phonetist-bot-main/
│
├── venv/                          # Виртуальное окружение Python
│   └── ...                        # Все установленные пакеты (aiogram, openai, numpy и т.д.)
│
├── .env                           # Файл с секретными ключами
│   ├── BOT_TOKEN=...              # Токен Telegram-бота
│   ├── OPENAI_API_KEY=...         # API ключ OpenAI
│   └── BASE_URL=...               # Базовый URL API (для Artemox)
│
├── requirements.txt               # Список зависимостей проекта
│   └── aiogram==3.4.1
│   └── openai>=2.15.0
│   └── numpy
│   └── python-dotenv
│   └── markdown
│   └── aiohttp, aiofiles
│
├── index.pkl                       # Индекс документов с эмбеддингами (pickle)
│   └── data = {
│           "documents": [...],     # Список текстов документов по фонетике
│           "embeddings": [...]     # NumPy массив эмбеддингов для каждого документа
│       }
│
├── build_index.py                  # Скрипт для создания эмбеддингов и сохранения их в index.pkl
│   
│
├── bot.py                          # Главный скрипт Tбота
│   └── Загружает index.pkl
│   └── Подключается к OpenAI API
│   └── Обрабатывает команды и сообщения через aiogram
│   └── Ищет по эмбеддингам с помощью косинусного сходства
│   └── Формирует читаемый ответ и отправляет пользователю
│
├── README.md                       # Инструкция по запуску проекта
│
└── knowledge_base/                      # Тексты для индексации
    └── фонетика_билеты.txt
    └── ...
