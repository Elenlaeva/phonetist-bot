# Telegram-бот «Фонетист»

## Описание
Это бот, который отвечает на вопросы
по курсу «Сегментные и супрасегментные единицы» на основе базы знаний,
сформированной из экзаменационных билетов.

## Функции
- прием текстовых вопросов в свободной форме;
- анализ запроса пользователя;
- поиск релевантной информации в базе знаний;
- генерация развернутого теоретического ответа с помощью LLM.

## Стек технологий
- Python
- Telegram Bot API
- aiogram
- LLaMA / аналогичная LLM
- Markdown (.md)
